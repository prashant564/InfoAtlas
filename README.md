# WorldExplorer

This project allows users to fetch details from all the countries in the world.

# Project Setup

Install all packages

```sh
yarn
```
For iOS, perform

```sh
cd ios
pod install
```

Finally, run the build

```sh
yarn run android
yarn run ios
```
